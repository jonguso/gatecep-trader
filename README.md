# GATECEP Trader Terminal UI

Broker-terminal style update inspired by common online trading workflows.

Includes:
- Dashboard
- Market Watch
- Order Entry
- Orders / Order Report
- Portfolio Valuation
- Available Funds
- Reports / Audit Log
- Risk Management
- Coach G
- Multi-symbol delayed/demo pricing
- Per-symbol order book and matching engine

## Run backend

```bash
cd backend
copy .env.example .env
npm install
npm start
```

## Run frontend

```bash
cd frontend
npm install
npm start
```

Open:

```txt
http://localhost:3000
```

This is a simulator, not connected to a licensed broker or live NSE execution.
