# GATECEP User-Friendly Design Patch

Copy this `mobile/` folder over your existing `gatecep-clean-full-package/mobile/`.

This patch adds:
- NSE design tokens
- reusable user-friendly UI components
- refreshed Home/Portfolio screen
- refreshed Trade ticket with fee summary
- KES formatting helpers

Run:

```bash
cd mobile
npx expo start -c
```
