\# Release 3.0.3 – Migration Sprint



\## Objective



Move existing GateCEP features onto the shared engines and shared constants created during GateCEP 3.0 foundation work.



\## Scope



\- ENG-PORT-001 adoption

\- ENG-MKT-001 adoption

\- Shared broker catalog adoption

\- Investor profile single source of truth

\- Static price removal

\- Duplicate logic identification



\## Not In Scope



\- New UI features

\- Real broker APIs

\- Live NSE production feed

\- Full reporting module



\## Success Criteria



\- Dashboard uses shared valuation logic.

\- Smart Portfolio uses market prices instead of hardcoded prices.

\- Broker screens use shared/constants/brokers.js.

\- Investor profile data persists through backend API.

\- Duplicated files are marked DEPRECATED or OBSOLETE.
